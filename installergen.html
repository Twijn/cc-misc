installergen

Installer Generator for CC-Misc Utilities Interactive tool to install ComputerCraft libraries with dependency management This tool provides a user-friendly interface to select and install libraries from the cc-misc repository directly to your ComputerCraft computer.

Installation

Recommended: Install via installergen (handles dependencies automatically):

wget run https://raw.githubusercontent.com/Twijn/cc-misc/main/util/installergen.lua installergen
View on GitHub →

This pattern downloads and runs the library at runtime, automatically installing it if not present:

-- Download and require installergen with automatic installation
if not fs.exists("installergen.lua") then
    shell.run("wget", "run", "https://raw.githubusercontent.com/Twijn/cc-misc/main/util/installergen.lua", "installergen")
end
local installergen = require("installergen")

-- Use the library
-- (your code here)

Alternative: Direct download via wget:

wget https://raw.githubusercontent.com/Twijn/cc-misc/main/util/installergen.lua
